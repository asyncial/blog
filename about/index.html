Software I use and recommend

Webbrowser

  • Vivaldi - My main browser. Very customizable and integrates excellent tab management. Completely controllable with your keyboard, if you want to.
  • Opera - My main browser on my phone, but the desktop browser integrates some pretty nice functionality. Sometimes very innovative features.
  • Firefox - If some site isn’t compatible with vivaldi or someone else is using my laptop.

Terminal Environment

  • urxvt - Pretty standard. Customizable through your .Xresources, which is nice.
  • zsh - Not the most minimalistic, but the most comfortable for me.

Communication

  • Thunderbird - Again, pretty standard, but GPG handling with Enigmail is just so convinient.
  • mcabber - Nice XMPP/Jabber Client for the terminal. Has support for OTR.
  • qTox - The Tox client, that comes across the most stable and reliable.
  • WhatsApp - You know it. But I actually think, it is a really good and secure messenger.
  • Signal - If you want to use a securely encrypted messenger use this one. Not that Telegram crap.
  • Silence - If you need encrypted SMS
  • K9-Mail - The android-mail-client, that is opensource

Entertainment

  • Foobar2000 - The music-player I loved. The reason, I changed to Linux like a year later, because I didn’t found a replacement.
  • Deadbeef - Foobar2000 for Linux. Not as customizable as the original, but it is alright.
  • mpv - Lightweight video player.
  • Poweramp - Best android music app. Hands down.

Work

  • LibreOffice - Standard. Don’t use OpenOffice, it has almost no active maintainers anymore.
  • IntelliJ IDEA - We are supposed to program Java in university. This is the best IDE I found.
  • NeoVim - I am not settled on this one. I like it so far, in fact I use it to write down these lines, but emacs or Lime Text are not out of the game.
  • qpdfview - PDF readers for linux suck. This is the least sucky, I found so far.
  • GIMP - Edit pictures.

Desktop

  • i3-gaps - My window manager. Didn’t find a better one yet. Really nice, when configured properly.
  • polybar - A nice bar for your DIY Desktop Environment.
  • rofi - Rice-able application launcher.
  • dunst - Your new, lightweight notification daemon.
  • Budgie - A nice, no bullshit Desktop Environment based on the GNOME applications. It is really nice and I like the communication of the developers.
  • KDE - Fully featured, highly customizable Desktop Environment.

Linux Distributions

  • Alpine Linux - Impressive, extremely minimalistic Distribution. Perfect for one purpose projects. Also security-hardened.
  • Arch Linux - Distro, that puts you in control. Not perfect, but for me, it is the best compromise between my idealism and my need for a reliable operating system.
  • Solus - Highly optimized, very good assembled distro. It is suitable for newcomers and it makes some effort to give the users a good experience with steam and games. I like it. I think their developer blog is interesting. I like Budgie very much.
  • OpenSUSE Leap - Very stable and polished distribution. It is the one, I usually recommend linux newcomers. It is a solid base, but gives you the possibility to explore the system, if you want to. It’s not dumbed down, like Ubuntu, but gives you the linux feel without needing you to be a technology enthusiast.
  • Puppy - The most reliable live distro I found so far. I use, if I broke on of my operating systems.
  • SliTaz - The best lightweight distro, which ships a graphical desktop. At least for me.
